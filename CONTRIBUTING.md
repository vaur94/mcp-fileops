# Contributing

> English | [Turkce](./CONTRIBUTING.tr.md)

Thanks for contributing to `mcp-fileops`.

## Contribution rules

- keep changes focused and reviewable
- do not copy source from `vaur94/mcpbase`
- preserve the deny-by-default filesystem model
- add or update tests for behavior changes
- keep English and Turkish documentation in sync when docs change

## Recommended workflow

1. install dependencies with `npm ci`
2. run `npm run ci:check` before opening a PR
3. update docs when tool behavior, config, or security semantics change
