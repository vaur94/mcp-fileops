# Destek

> [English](./SUPPORT.md) | Turkce

Normal urun ve dokumantasyon destegi icin deponun herkese acik GitHub kanallarini kullanin.

## Nereden destek alinmali

- tekrar uretilebilir bug veya dokuman problemleri icin GitHub issue acin
- repository ileride discussions etkinlestirirse genel tartismalari orada yurutun
- somut dokuman veya kod iyilestirmeleri icin pull request kullanin

## Neler eklenmeli

- isletim sistemi ve Node.js surumu
- npm surumu
- gerekliyse MCP istemci adi
- kullandiginiz komut, config veya workflow
- mumkunse tam hata ciktisi

## Guvenlik sorunlari

Istismar edilebilir guvenlik bulgularini normal public issue ile bildirmeyin. Bunun yerine `SECURITY.md` icindeki private reporting yonergelerini kullanin.

## Kapsam notu

Bu depoda su anda ayri bir ticari destek kanali veya SLA dokumante edilmis degildir.
