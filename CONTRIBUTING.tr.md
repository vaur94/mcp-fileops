# Katki Rehberi

> [English](./CONTRIBUTING.md) | Turkce

`mcp-fileops` projesine katkida bulundugunuz icin tesekkurler.

## Katki kurallari

- degisiklikleri odakli ve incelemesi kolay tutun
- `vaur94/mcpbase` kaynak kodunu kopyalamayin
- varsayilan olarak kapali dosya sistemi modelini koruyun
- davranis degisikligi varsa test ekleyin veya guncelleyin
- dokumantasyon degistiginde Ingilizce ve Turkce dosyalari birlikte guncelleyin

## Onerilen akis

1. `npm ci` ile bagimliliklari kurun
2. PR acmadan once `npm run ci:check` calistirin
3. arac davranisi, config veya guvenlik semantigi degisirse ilgili dokumanlari guncelleyin
